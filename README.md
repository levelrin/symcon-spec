# Symcon

It's a symbolic notation for economics.

The goal is to write a sentence about economics.

The problem is that the sentence can be unnecessarily long in plain English.

Although a mathematical expression may provide a concise and accurate documentation, it can be unintuitive.

Therefore, it might be better to design a new writing system for concise and intuitive documentation.

## Fixed Variable

"The price is fixed." can be written as `🔒price`.

## Scope

Parentheses can be used for a variable name with spaces.

For example, "The stock of money is fixed." can be written as `🔒(stock of money)`.
